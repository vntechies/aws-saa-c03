| Benefit | Description | Cost Impact |
|---------|-------------|-------------|
| **No Servers** | No EC2 instances to manage | Pay per request |
| **Auto-Scaling** | Automatic scaling to millions of requests | No over-provisioning |
| **High Availability** | Built-in Multi-AZ | No additional cost |
| **Low Latency** | CloudFront + Lambda@Edge | Global performance |

### Cost Comparison

**Traditional (EC2):**
```
2 x t3.medium (24/7) = $60/month
RDS t3.small (24/7) = $30/month
ALB = $22/month
Total: ~$112/month (minimum, always running)
```

**Serverless:**
```
S3 hosting = $1/month
Lambda (1M requests) = $0.20
API Gateway (1M requests) = $3.50
DynamoDB (On-Demand) = $1.25
Total: ~$6/month (pay per use)
```

**Savings:** ~95% for low-traffic applications
